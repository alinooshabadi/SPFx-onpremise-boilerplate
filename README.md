## Minimal Path to Awesome

- Clone this repository
- Ensure that you are at the solution folder
- in the command-line run:

  - **npm install**
  - **npm run fix**
  - **gulp serve**

- For Deployment, run:
  - **gulp deploy --env test --debug**

> Include any additional steps as needed.
